Инструкция по установке

1 Распакуйте архив
Windows: Правый клик → 7-Zip → Extract Here
Linux/Mac: tar -xzf browser_extension.tar.gz
2 Откройте Chrome Extensions
Введите в адресной строке: chrome://extensions/
3 Включите Developer mode
Переключатель в правом верхнем углу → ON
4 Загрузите расширение
Нажмите "Load unpacked" → Выберите папку browser_extension/
5 Готово!
Расширение установлено и готово к использованию ✅
Как использовать
  1. Откройте czbooks.net
  2. Дождитесь Cloudflare challenge (~5 сек)
  3. Кликните на расширение
  4. Нажмите "Извлечь Cookies"
  5. Нажмите "Отправить в Web App"
Преимущества
  • Автоматическое извлечение cookies
  • Отправка в Web App одной кнопкой
  • 5 секунд вместо 10
  • 2 клика вместо 7 шагов
Безопасность
  • Работает только локально
  • Исходный код открыт
  • Не отправляет данные наружу
  • Cookies только в ваш Web App
